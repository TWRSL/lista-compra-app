# 🛒 Lista de la Compra Familiar

App web progresiva para gestionar la lista de la compra compartida entre miembros de la familia, con soporte para múltiples supermercados.

## Supermercados

* Mercadona
* Ahorramás
* ALDI
* (ampliable: Carrefour, Alcampo...)

## Tecnología

* Frontend: HTML + CSS + JavaScript (un solo archivo)
* Base de datos y auth: Supabase
* Hosting: GitHub Pages)

## Funcionalidades

\- Lista compartida en tiempo real

\- Tabs por supermercado

\- Estados: Falta → Asignado → Comprado

\- Acceso por invitación del admin
